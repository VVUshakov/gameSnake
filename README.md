# Змейка (Snake)

Классическая игра «Змейка», написанная на C# для консоли.
Проект создан в учебных целях для демонстрации поэтапной разработки.

## Управление
W / ↑ - движение вверх
S / ↓ - движение вниз
A / ← - движение влево
D / → - движение вправо
Esc - выход из игры
- Escape — выход из игры

## Правила игры
- Змейка движется непрерывно в заданном направлении
- Съедайте еду (●), чтобы увеличить длину и счёт
- Не врезайтесь в стены и в собственное тело
- Чем длиннее змейка, тем сложнее играть

## Особенности реализации
- Чистая архитектура с интерфейсами
- Независимость от платформы (можно заменить консоль на графику)
- Пошаговая разработка в Git (23 ветки)
- Соблюдение принципов SOLID
- Элегантное математическое построение змейки
- Автоматическое центрирование змейки на поле
- Умное создание еды с проверкой свободного места
- Обработка ситуации "нет места для еды"
